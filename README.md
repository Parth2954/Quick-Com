# 🛒 Quick-Commerce Grocery SPA (AngularJS)

A Single Page Application (SPA) inspired by modern quick-commerce platforms like Blinkit, Zepto, and Instamart. Built using **AngularJS, HTML, and CSS**, this project simulates an online grocery shopping experience with smooth navigation, cart management, and user authentication.

---

## 🚀 Features

- 🏠 Home page with product listings
- 🔍 Live search and category filtering
- 🛒 Add to cart / remove / update quantity
- 💳 Checkout system with order summary
- 🔐 User authentication (Register & Login)
- 💾 Data persistence using localStorage
- 🔄 Single Page Application (no page reloads)
- 📱 Responsive and modern UI design

---

## 🧱 Tech Stack

- AngularJS (1.x)
- HTML5
- CSS3
- JavaScript (ES5)

## ⚙️ How to Run Locally

1. Open project folder
2. Open terminal inside the folder
3. Run:
   ```bash
   python -m http.server 8000
   ```
4. Open browser:
   ```
   http://localhost:8000
   ```

## 🔐 Authentication

- Users can register using email and password
- Login system validates stored users
- Session maintained using localStorage

---

## 📦 Data Handling

- Products are stored in static JSON format
- Cart and user data stored in browser localStorage
